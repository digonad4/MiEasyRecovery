   Este arquivo contém o gerenciador de arquivos multifuncional WinRAR.

   Recursos do WinRAR:

 * O WinRAR introduz um algoritmo de compressão original. Ele fornece
    altas taxas de compressão em arquivos executáveis, bibliotecas de 
    objeto, grandes arquivos de texto, etc.

 * O formato interno do RAR não limita a compressão e o tamanho do arquivo 
   bem como um número de arquivos no arquivo comprimido. Estes valores podem 
   ser limitados pelo arquivo e pelo sistema operacional como também pela 
   memória disponível.

 * O WinRAR fornece suporte completo para arquivos RAR e ZIP 2.0 e pode
   descomprimir arquivos como CAB, ARJ, LZH, TAR, GZ, ACE, UUE, BZ2, JAR,
   ISO, XZ, Z, 7Z.

 * O WinRAR oferece novas facilidades e funções como uma interface gráfica 
   interativa e uma interface de linha de comandos.

 * O WinRAR fornece a funcionalidade de criar arquivos 'sólidos', que
   poderão aumentar a taxa de compressão entre 10% - 50% a mais sobre
   os métodos mais comuns, particularmente quando se comprime um grande 
   número de pequenos arquivos.

 * O WinRAR pode criar e alterar arquivos SFX personalizáveis.

 * O WinRAR pode criar arquivos de volumes múltiplos como SFX.

 * O registro de recuperação opcional permite proteger arquivos de danos.

 * O WinRAR oferece um número de funções de serviço, tais como criptografia 
   de arquivos, adicionar comentários para arquivos comprimidos e preservar 
   a segurança de arquivos NTFS.
